# Tell-App
Tell è un applicazione desktop che consente di gestire la vendita di biglietti e la diffusione di informazioni riguardo eventi teatrali, specialmente concerti e opere liriche. È il progetto finale di gruppo per il corso di [Ingegneria del Software](https://guide.univpm.it/af.php?lang=lang-ita&af=247892) all'Università Politecnica delle Marche in Ancona.

## Come si usa?
Per eseguire Tell è necessario avere installato il linguaggio Python e il package PyQt6, installabile tramite pip.
Per avviare l'applicazione è sufficiente clonare la repository, posizionarsi nella cartella `codice` ed eseguire il file Python "main.py". Se non vengono passati parametri, Tell utilizzerà il percorso di default `./db`, ovvero `codice/db`, per memorizzare il database. La repository fornisce, a questo percorso, una cartella già riempita con un database di esempio. È possibile passare un singolo parametro per specificare un'altro percorso per la cartella del database.
